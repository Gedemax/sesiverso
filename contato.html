Aviso de Privacidade

Sua privacidade é importante para nós. Este formulário coleta apenas as informações necessárias para responder sua mensagem.

O que coletamos:

  • Nome (opcional) - apenas para personalizar nossa resposta
  • E-mail (opcional) - apenas se você quiser receber nossa resposta
  • Mensagem - para entendermos sua dúvida ou sugestão

O que NÃO coletamos:

  • Dados pessoais sensíveis (CPF, idade exata, endereço)
  • Informações sobre saúde mental específicas
  • Dados de menores de idade sem consentimento dos responsáveis

Menores de 18 anos: Se você tem menos de 18 anos, peça para um responsável ler este aviso e autorizar o envio da mensagem.

Uso dos dados: Suas informações serão usadas apenas para responder sua mensagem e poderão ser citadas anonimamente em futuras pesquisas (sem identificação pessoal).

Perguntas Frequentes

Como posso colaborar com o projeto?

Existem várias formas de colaborar:

  • Compartilhando o site com outros jovens e educadores
  • Implementando as intervenções propostas em sua escola
  • Enviando feedback sobre nossos materiais
  • Conectando-nos com outros pesquisadores da área

Use o formulário acima selecionando "Proposta de colaboração" para mais detalhes.

Posso usar os materiais do projeto em minha escola?

Sim! Todos os nossos materiais estão licenciados sob Creative Commons e podem ser usados livremente para fins educacionais. Apenas pedimos que:

  • Dê crédito ao projeto Sesiverso
  • Não use para fins comerciais
  • Mantenha as informações de contato de ajuda (CVV 188)

Veja mais detalhes na página da galeria.

Como foram coletados os dados da pesquisa?

Nossa pesquisa seguiu todos os protocolos éticos para pesquisas com jovens:

  • Questionários anônimos online
  • Consentimento dos responsáveis para menores de 18 anos
  • Nenhum dado pessoal identificável foi coletado
  • Todos os participantes foram informados sobre recursos de apoio

Mais detalhes na metodologia do artigo.

O projeto continuará após a feira de ciências?

Sim! Nossos planos incluem:

  • Manter o site atualizado com novos recursos
  • Implementar as intervenções em mais escolas
  • Continuar coletando dados para pesquisas futuras
  • Expandir para outras questões de saúde mental digital

Entre em contato se quiser acompanhar nossos próximos passos!

Como posso citar este projeto em trabalhos acadêmicos?

Para citar o projeto Sesiverso:

Para trabalhos acadêmicos mais formais, entre em contato conosco para receber informações adicionais sobre a pesquisa.

Outras Formas de Contato

Redes Sociais Educacionais

Seguimos as diretrizes de segurança para menores de idade. Não mantemos perfis em redes sociais para este projeto, priorizando a comunicação direta e segura através deste site.

Apresentações e Eventos

Estamos disponíveis para apresentar nossa pesquisa em escolas, eventos acadêmicos e congressos de saúde mental. Use o formulário acima para solicitar uma apresentação.

Parcerias Institucionais

Buscamos parcerias com escolas, universidades, ONGs e instituições de saúde para expandir o impacto do projeto. Entre em contato para discutir colaborações.
